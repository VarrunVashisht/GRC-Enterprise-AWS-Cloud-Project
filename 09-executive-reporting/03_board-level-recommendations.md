# Board-Level Strategic Recommendations  
## Southern Cross Capital Pty Ltd  
### Cloud Security Governance Enhancement Plan

---

# 1️⃣ Strengthen IAM Governance

- Conduct quarterly access reviews  
- Enforce phishing-resistant MFA  
- Implement Privileged Access Management (PAM)  
- Apply just-in-time administrative access  

Rationale: Identity compromise remains highest enterprise risk.

---

# 2️⃣ Continuous Cloud Monitoring

- Implement automated anomaly detection  
- Deploy advanced threat detection (e.g., GuardDuty)  
- Validate alerts through periodic testing  

Rationale: Early detection reduces financial and regulatory impact.

---

# 3️⃣ Incident Response Testing

- Conduct annual tabletop simulation  
- Perform ransomware recovery test  
- Validate RTO and RPO objectives  

Rationale: Regulatory expectations require tested resilience.

---

# 4️⃣ Third-Party Risk Review

- Conduct vendor security assessments  
- Evaluate API integration security  
- Review third-party access controls  

Rationale: Supply chain risk can bypass internal controls.

---

# 5️⃣ Security Awareness Training

- Conduct quarterly phishing simulations  
- Mandatory annual cybersecurity training  
- Executive cyber awareness briefing  

Rationale: Human error remains primary attack vector.

---

# Strategic Summary

Cybersecurity should remain a standing board agenda item.

Recommended focus areas:

Identity  
Resilience  
Monitoring  
Third-party governance  

Continued executive oversight ensures alignment with regulatory obligations and protects customer trust.
